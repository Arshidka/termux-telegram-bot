# Termux Telegram Bot

This is a telegram bot using this library [telegraf](telegraf.js.org)

# How To Install ?
 So easy just follow this tutorial!
```cmd
-> pkg install git
-> git clone https://github.com/Xlaaf/termux-telegram-bot
-> cd termux-telegram-bot
-> pkg install python
-> python main.py
```
